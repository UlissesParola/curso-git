Git

Trabalhando com Git e Github

Principais funcionalidades e comandos:

  • git init

    dentro de um diretório para vincula-lo ao git.

  • git status

    para saber os status dos arquivos.

  • git add "nome do arquivo>

    para adicionar um arquivo ao monitoramento do git.

  • git commit -m

    para enviar as alterações para o repositório juntamente com uma mensagem curta. Sem a flag -m será aberto o editor de texto padrão (nano, vim...) para que seja incluída a mensagem.
